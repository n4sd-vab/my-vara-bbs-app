    G8BPQ BBS Command Reference

    
A - Abort Output 
B - Logoff 
HOMEBBS - Display or SET your HomeBBS address 
INFO - Display information about this BBS 
I CALL - Lookup CALL in WP Allows *CALL CALL* *CALL* wildcards 
I@ PARAM - Lookup @BBS in WP 
IZ PARAM - Lookup Zip Codes in WP 
IH PARAM - Lookup HA elements in WP - eg USA EU etc 
K - Kill Message(s) 
K  msg#  
KM (Kill my read messages) 
L - List Message(s) -  
L = List New, LR = List New (Oldest first) 
LM = List Mine, L> Call, L< Call, L@ = List to, from or at 
LL num = List msg num, L num-num = List Range 
LN LY LH LK LF L$ LD = List Message with corresponding Status 
LB LP LT = List Message with corresponding Type 
LC = List TO fields of all active bulletins 
You can combine most selections eg LMP, LMN LB< G8BPQ 
LISTFILES or FILES - List files available for download 
N Name - Set Your Name 
NODE - Return to Node 
OP n - Set Page Length (Output will pause every n lines) 
PASS Password - Set BBS Password 
Q QTH - Set Your QTH 
R - Read Message(s)  
R num 
RM (Read new messages to me),  
RMR (RM oldest first) 
READ Name - Read File 
S - Send Message 
S or SP Send Personal,  
SB Send Bull,  
ST Send NTS,  
SR Num - Send Reply,  
SC Num - Send Copy 
X - Toggle Expert Mode 
YAPP - Download file from BBS using YAPP protocol

    About My Vara BBS App

    This is my Electron app to connect to a G8BPQ Packet BBS 
via VARA FM. It is built using Electron and the VARA API, 
and is designed to provide a user-friendly interface for 
accessing BBS features like message lists, reading messages, 
sending messages, YAPP file transfers, and an address book.
